#+title: hyprland-zen native host

* Components

- ~hyprland-zen-host~
  - line-delimited JSON bridge used by Emacs (~hyprland-zen.el~)
  - listens on unix socket and forwards messages between Emacs and browser host
- ~hyprland-zen-native-host~
  - Native Messaging adapter launched by Zen/Firefox
  - translates Native Messaging frames to unix socket line protocol

Both scripts use this socket path:

- ~HYPRLAND_ZEN_SOCKET~ env var when set
- else ~$XDG_RUNTIME_DIR/hyprland-zen-bridge.sock~
- else ~/tmp/hyprland-zen-bridge-<uid>.sock~

* Install notes

1. Put both scripts on your PATH (or use absolute paths)
2. Copy ~hyprland_zen_bridge.json~ to your browser's NativeMessagingHosts dir
3. In that JSON file, set ~path~ to absolute path of ~hyprland-zen-native-host~
4. Ensure ~allowed_extensions~ includes extension id from ~manifest.json~

* Emacs side

Default ~hyprland-zen-host-command~ is ~("hyprland-zen-host")~.
If not on PATH, you can set:

#+begin_src emacs-lisp
(setq hyprland-zen-host-command '("/absolute/path/to/hyprland-zen-host"))
#+end_src
